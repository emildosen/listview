# ListView

A lightweight, open-source web app for managing data stored in SharePoint lists. Built for simple CRM or information knowledge use cases.

## Features

- SharePoint list storage with M365 authentication
- Modular design - configure lists without code changes
- Custom rollup views and reporting
- Pure client-side (no backend required)

## Tech Stack

- React + Vite
- MSAL React (authentication)
- Microsoft Graph API

## Quick Start

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and configure your Entra app credentials.
